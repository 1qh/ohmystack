## Prerequisites

<Tab value="Convex">

- A [Convex](https://convex.dev) account (free tier works)

</Tab>
<Tab value="SpacetimeDB">

- [Docker](https://docs.docker.com/get-docker/) (for SpacetimeDB + MinIO)
- [Bun](https://bun.sh) >= 1.1
- [SpacetimeDB CLI](https://spacetimedb.com/install) v2.0+
- Node.js >= 20 (for Next.js)

Install the CLI:

```bash
curl -sSf https://install.spacetimedb.com | sh
```

</Tab>

## 1. Scaffold the project

`noboil init` clones a fully-wired template, removes the parts you don't need, patches `package.json`, and runs `bun install`. It must target an empty directory.

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

```bash
bunx noboil@latest init my-app --db=convex
cd my-app
```

</Tab>
<Tab value="SpacetimeDB">

```bash
bunx noboil@latest init my-app --db=spacetimedb
cd my-app
```

</Tab>
</Tabs>

Pass `--no-demos` to skip the demo apps, or `--skip-install` to skip `bun install`. With both flags plus a target directory, `init` runs non-interactively.

## 2. Start the backend

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

The template includes a `backend/convex` workspace with `t.ts` (Zod schemas), `schema.ts` (`defineSchema` registrations), `lazy.ts` (your `setup()` call), and one example module per table (`blog.ts`, `chat.ts`, …) that exports CRUD endpoints from a single `crud()` call.

Boot Convex:

```bash
bunx convex dev
```

Authentication is wired through `@convex-dev/auth`. See the [Convex Auth guide](https://docs.convex.dev/auth) to configure providers — every CRUD factory enforces auth by default.

</Tab>
<Tab value="SpacetimeDB">

The template ships `spacetimedb.yml` (Docker Compose) and a `backend/spacetimedb` workspace with `t.ts` (Zod schemas wrapped with `schema()`) and `src/index.ts` (the `noboil()` module definition). Boot SpacetimeDB + MinIO and publish the module:

```bash
bun spacetime:up           # Docker Compose: SpacetimeDB on :4000, MinIO on :4600
bun spacetime:publish      # publish module
bun spacetime:generate     # generate TypeScript bindings
```

Re-run `spacetime:generate` after any schema change.

</Tab>
</Tabs>

## 4. Define your schema

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

`noboil init` generates `convex/t.ts` with your Zod schemas. Edit it to define your tables:

```typescript
import { makeOwned } from '@noboil/convex/schema'
import { boolean, object, string } from 'zod/v4'

const owned = makeOwned({
  blog: object({
    title: string().min(1),
    content: string().min(3),
    category: string(),
    published: boolean()
  })
})

export { owned }
```

</Tab>
<Tab value="SpacetimeDB">

Define shared Zod schemas in `t.ts`, then wire everything together with `noboil()` in `src/index.ts`.

**`t.ts`** — field definitions:

```typescript
import { schema } from '@noboil/spacetimedb/schema'
import { boolean, object, string } from 'zod/v4'

const s = schema({
  owned: {
    post: object({ title: string(), content: string() })
  },
  singleton: {
    profile: object({ displayName: string(), bio: string().optional() })
  }
})

export { s }
```

**`src/index.ts`** — backend module:

```typescript
import { noboil } from '@noboil/spacetimedb/server'
import { s } from '../t'

export default noboil(({ table }) => ({
  post: table(s.post, { pub: 'published' }),
  profile: table(s.profile)
}))
```

`noboil()` builds the schema, registers CRUD reducers, and exports the module. System fields (`id`, `updatedAt`, `userId`) are added automatically.

</Tab>
</Tabs>

## 5. Run the dev server

```bash
bun dev
```

Open the demo app on the port printed in the terminal. The backend is already live from step 2.

## 6. Connect from React

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

Wrap your app with `ConvexProvider` in `app/providers.tsx`:

```tsx
'use client'
import { ConvexProvider, ConvexReactClient } from 'convex/react'

const convex = new ConvexReactClient(process.env.NEXT_PUBLIC_CONVEX_URL!)

export const Providers = ({ children }: { children: React.ReactNode }) => (
  <ConvexProvider client={convex}>{children}</ConvexProvider>
)
```

</Tab>
<Tab value="SpacetimeDB">

Wrap your app with `SpacetimeDBProvider` in `app/providers.tsx`:

```typescript
'use client'

import { SpacetimeDBProvider } from 'spacetimedb/react'
import { createSpacetimeClient, createTokenStore } from '@noboil/spacetimedb/react'
import { DbConnection } from '@/generated/module_bindings'

const SPACETIMEDB_URI = process.env.NEXT_PUBLIC_SPACETIMEDB_URI ?? 'ws://localhost:4000'
const MODULE_NAME = process.env.NEXT_PUBLIC_MODULE_NAME ?? 'my-app'
const tokenStore = createTokenStore()

export const Providers = ({ children }: { children: React.ReactNode }) => (
  <SpacetimeDBProvider
    uri={SPACETIMEDB_URI}
    module={MODULE_NAME}
    createConnection={() =>
      createSpacetimeClient({
        DbConnection,
        moduleName: MODULE_NAME,
        tokenStore,
        uri: SPACETIMEDB_URI,
      })
    }
  >
    {children}
  </SpacetimeDBProvider>
)
```

## 7. Query data

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

```tsx
'use client'
import { useList } from '@noboil/convex/react'
import { api } from '../convex/_generated/api'

const BlogList = () => {
  const { items, loadMore, status } = useList(api.blog.list)
  return (
    <ul>
      {items.map(b => (
        <li key={b._id}>{b.title}</li>
      ))}
      {status === 'CanLoadMore' && (
        <button onClick={loadMore}>Load more</button>
      )}
    </ul>
  )
}

export default BlogList
```

</Tab>
<Tab value="SpacetimeDB">

`useTable` subscribes to a table and returns all rows in real-time:

```typescript
'use client'

import { useTable } from 'spacetimedb/react'
import { tables } from '@/generated/module_bindings'
import { useList } from '@noboil/spacetimedb/react'

const PostList = () => {
  const [posts, isReady] = useTable(tables.post)
  const { data, hasMore, loadMore } = useList(posts, isReady, {
    sort: { field: 'updatedAt', direction: 'desc' },
    pageSize: 20,
  })

  if (!isReady) return <div>Loading...</div>

  return (
    <ul>
      {data.map(post => (
        <li key={post.id}>{post.title}</li>
      ))}
      {hasMore && <button onClick={loadMore}>Load more</button>}
    </ul>
  )
}

export default PostList
```

## 8. Write data

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

```tsx
import { useMutation } from 'convex/react'
import { api } from '../convex/_generated/api'

const CreateBlog = () => {
  const create = useMutation(api.blog.create)
  return (
    <button
      onClick={() =>
        create({
          title: 'Hello',
          content: 'World',
          category: 'tech',
          published: true
        })
      }
    >
      New post
    </button>
  )
}
```

</Tab>
<Tab value="SpacetimeDB">

`useReducer` returns a function that calls a server-side reducer:

```typescript
'use client'

import { useReducer } from 'spacetimedb/react'
import { reducers } from '@/generated/module_bindings'

const CreatePost = () => {
  const createPost = useReducer(reducers.create_post)

  const handleSubmit = async (e: React.FormEvent<HTMLFormElement>) => {
    e.preventDefault()
    const form = new FormData(e.currentTarget)
    await createPost({
      title: form.get('title') as string,
      content: form.get('content') as string,
      published: false,
    })
  }

  return (
    <form onSubmit={handleSubmit}>
      <input name="title" placeholder="Title" required />
      <textarea name="content" placeholder="Content" required />
      <button type="submit">Create post</button>
    </form>
  )
}

export default CreatePost
```

## 9. Typesafe forms

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

```tsx
import { Form } from '@noboil/convex/components'
import { useFormMutation } from '@noboil/convex/react'
import { api } from '../convex/_generated/api'
import { owned } from '../convex/t'

const BlogForm = () => {
  const form = useFormMutation(api.blog.create, owned.blog)
  return (
    <Form {...form}>
      {({ Text, Toggle, Choose }) => (
        <>
          <Text name="title" />
          <Text name="content" />
          <Choose name="category" />
          <Toggle name="published" />
          <button type="submit">Create</button>
        </>
      )}
    </Form>
  )
}
```

</Tab>
<Tab value="SpacetimeDB">

```typescript
'use client'

import { useForm } from '@tanstack/react-form'
import { zodValidator } from '@tanstack/zod-form-adapter'
import { useReducer } from 'spacetimedb/react'
import { reducers, tables } from '@/generated/module_bindings'
import { zodFromTable } from '@noboil/spacetimedb'

const createPostSchema = zodFromTable(tables.post.columns, {
  exclude: ['published']
})

const BlogForm = () => {
  const createPost = useReducer(reducers.create_post)

  const form = useForm({
    defaultValues: { title: '', content: '' },
    validatorAdapter: zodValidator(),
    validators: { onChange: createPostSchema },
    onSubmit: async ({ value }) => {
      await createPost({ ...value, published: false })
      form.reset()
    },
  })

  return (
    <form onSubmit={e => { e.preventDefault(); form.handleSubmit() }}>
      <form.Field
        name="title"
        children={field => (
          <div>
            <label>Title</label>
            <input
              value={field.state.value}
              onChange={e => field.handleChange(e.target.value)}
              onBlur={field.handleBlur}
            />
          </div>
        )}
      />
      <form.Field
        name="content"
        children={field => (
          <div>
            <label>Content</label>
            <textarea
              value={field.state.value}
              onChange={e => field.handleChange(e.target.value)}
              onBlur={field.handleBlur}
            />
          </div>
        )}
      />
      <button type="submit" disabled={form.state.isSubmitting}>
        {form.state.isSubmitting ? 'Creating...' : 'Create'}
      </button>
    </form>
  )
}
```

## What you get for free

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

From a single `crud('blog', owned.blog)` call:

- `list` — paginated query with `where` clause support
- `read` — single doc by ID with ownership check
- `create` — validated insert with auth + rate limiting
- `update` — partial patch with conflict detection
- `rm` — ownership-checked delete with file cleanup

</Tab>
<Tab value="SpacetimeDB">

From a single `table(s.post)` call, `noboil()` generates:

- `create_post` — validated insert with auth check
- `update_post` — partial update with conflict detection via `expectedUpdatedAt`
- `rm_post` — ownership-checked delete
- `get_profile` / `upsert_profile` — for singleton tables

Updates from any client appear in all connected clients within ~39ms (local Docker).

</Tab>
</Tabs>

## TypeScript type inference

<Tabs groupId="db" items={['Convex', 'SpacetimeDB']} persist>
<Tab value="Convex">

The generated `api` object is fully typed. `api.blog.create` knows exactly which fields are required and their types — passing a wrong field name is a compile error.

</Tab>
<Tab value="SpacetimeDB">

`@noboil/spacetimedb` exports utility types that derive TypeScript types directly from your schema. No manual type duplication needed:

```typescript
import type { InferRow, InferCreate, InferUpdate } from '@noboil/spacetimedb'

type PostRow = InferRow<typeof postSchema>
type PostCreate = InferCreate<typeof postSchema>
type PostUpdate = InferUpdate<typeof postSchema>

const PostCard = ({ post }: { post: PostRow }) => <h2>{post.title}</h2>
```

For reducer types:

```typescript
import type { InferReducerArgs, InferReducerReturn } from '@noboil/spacetimedb'
import { reducers } from '@/generated/module_bindings'

type CreateArgs = InferReducerArgs<typeof reducers.create_post>
type CreateReturn = InferReducerReturn<typeof reducers.create_post>
```
